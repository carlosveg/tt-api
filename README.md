# TT - API

Aquí va a ir la documentación para levantar el proyecto

1. Clonar proyecto
2. `yarn install`
3. Renombrar el archivo `.env.template` a `.env`
4. Asignar las variables de entorno
5. Levantar la base de datos

```
docker compose up -d
```

6. Levantar `yarn start:dev`
